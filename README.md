## 当前学习目标|Current Learning Objectives
**1.电气工程核心基础**
 **精通C语言**：函数、数组、指针、内存管理、数据结构、算法（从易到难，详细且具体）。
 **电路与电子技术**：电路原理、模拟电路、数字电路等。
 
**2.嵌入式系统开发**
 **STM32单片机开发**：掌握HAL库，精通GPIO、定时器、中断、ADC、通信协议（UART,I2C,SPI）等核心外设。
 **嵌入式实时操作系统**：FreeRTOS。
 
**3.工程与科研工具链**
 **Python**：专注于**数据分析、科学计算与机器学习**，为数学建模和科研数据处理打下基础。
 **MATLAB/Simulink**:用于**控制系统设计、算法仿真与信号处理**。
 **Git**：版本控制与项目管理。
 **Altium Designer**:了解电路板设计流程。
 
**4.数学与算法**
 **数学建模**：掌握常用模型与算法。
 **数据结构与算法**：LeetCode/OJ平台定期练习。
